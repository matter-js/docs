This indicates a program within an electronic program guide (EPG).

Type declaration

  • audioLanguages: OptionalFieldType<string[]>

    This field shall indicate the audio language for the specific program. The value is a string containing one of the standard Tags for Identifying Languages RFC 5646. This field is optional but shall be provided if known.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.8

  • castList: OptionalFieldType<TypeFromFields<{
        name: FieldType<string>;
        role: FieldType<string>;

    This field shall represent a list of the cast or the crew on the program. A single cast member may have more than one role. This field is optional but shall be provided if known.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.18

  • categoryList: OptionalFieldType<TypeFromFields<{

    MatterSpecification.v13.Cluster § 6.6.5.7.19

  • identifier: FieldType<string>

    This field shall indicate a unique identifier for a program within an electronic program guide list. The identifier shall be unique across multiple channels.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.1

  • parentalGuidanceText: OptionalFieldType<string>

    This field shall represent a string providing additional information on the parental guidance. This field is optional.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.14

  • posterArtUrl: OptionalFieldType<string>

    This field shall represent a url of a poster that clients can use to render an image for the program on the detail view.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.11

  • ratings: OptionalFieldType<string[]>

    This field shall be used for indicating the level of parental guidance recommended for of a particular program. This can be any rating system used in the country or region where the program is

    broadcast. For example, in the United States “TV-PG” may contain material that parents can find not suitable for younger children but can be accepted in general for older children. This field is optional but shall be provided if known.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.9

  • recordingFlag: OptionalFieldType<TypeFromPartialBitSchema<{

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.15

  • releaseDate: OptionalFieldType<string>

    This field shall be a string, in ISO 8601 format, representing the date on which the program was released. This field is optional but when provided, the year shall be provided as part of the string.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.13

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.16

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.3

    This field shall indicate the subtitle for the specific program. For example, “Maybe Today" which is an episode name for “MCIS: Los Angeles”. This field is optional but shall be provided if applicable and known.

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.6

  • thumbnailUrl: OptionalFieldType<string>

    See

    MatterSpecification.v13.Cluster § 6.6.5.7.10

    See
